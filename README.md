# NexaFlow — Landing Page SaaS

Landing page fictícia para um SaaS chamado **NexaFlow**, criada para portfólio e demonstração de habilidades em **UI Design** e **Desenvolvimento Frontend**.

## 🚀 Objetivo
Este projeto foi desenvolvido para mostrar:
- Criação de interfaces modernas e responsivas.
- Uso de tipografia e identidade visual consistente.
- Estrutura de landing page com foco em conversão (CTA, planos, demonstração técnica).

## 🛠️ Tecnologias Utilizadas
- **HTML5** e **CSS3** (sem frameworks externos).
- **Google Fonts**: Syne e DM Mono.
- **Design responsivo** com media queries.
- **Animações CSS** para transições suaves.

## 📂 Estrutura da Página
- **Navbar fixa** com logo e links de navegação.
- **Hero Section** com título, descrição, estatísticas e botões de ação.
- **Features Section** com cards de funcionalidades.
- **Demo Section** com terminal interativo e stack técnica.
- **Pricing Section** com três planos (Starter, Pro, Enterprise).
- **CTA Banner** para conversão final.
- **Footer** com créditos e direitos.
